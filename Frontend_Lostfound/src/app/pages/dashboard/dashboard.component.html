Gestión de cuenta

Si deseas modificar tu perfil debes ingresar al apartado de gestión de cuenta. Se recomienda cambiar la clave temporal por una clave propia para mayor seguridad.

Valoración de usuarios

La página LostFound permitirá crear una valoración a un usuario que realiza una publicación. Si la persona tiene una valoración promedio menor a 3 estrellas, no podrá publicar durante un mes.
